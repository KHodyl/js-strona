JavaScript
JavaScript należy do języków skryptowych programowania. Obok języków HTML i CSS jest jednym z głównych języków stosowanych do tworzenia stron WWW. chodz JavaScript używa się najczęściej do operacji "Client-Side" na stronach, to nie limtuje się jego użycia tylko do przeglądarek.



Przykłady zastosowania JS na stronach:

- Okna "alert" / "pop-up" na stronie

- Obliczenia liczbowe

- "Tworzenie" liczb losowych / pseudo-losowych.

- Monitorowanie zachowiań użytkownika

- Animacje na stronie

- Interakcje z użytkownikiem

- Tworzenie plików "Cookies"



Inne zastosowania JS:

- Silniki JavaScript używane w różnych programach, by umożliwić zastosowanie komend JS

- Systemy wbudowane posiadające możliwość wykożystywania JS, w tym:

-- Robotyka

-- "Internet Rzeczy"
